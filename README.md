# SELMA
This repository contains the code for SELMA, our semantic and local code search system. 

Please note that this branch is currently outdated. New features are developed on the branch [30-implement-self-indexing-feature](https://github.com/AnReu/selma_code/tree/30-implement-self-indexing-feature). 

## Dependencies
You need to have Java 11+ installed on your system in order to use Terrier.
